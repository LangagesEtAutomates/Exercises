\begin{exercice}[Expressions rationnelles] \label{exo:lexing/regexp/write_regexp}

  Proposer des expressions rationnelles définissant les langages suivants :
  \begin{question}
  \item $L_1 = \{ab\}$
  \item $L_2 = \{a^nba^m \mid n, m \in \mathbb{N}\}$
  \item $L_3 = $ les identificateurs en C, composés de chiffres, lettres, et $\_$, mais ne commençant pas par un chiffre
  \item $L_4 = $ les nombres décimaux sans 0 inutiles
  \item $L_5 = $ les mots sur $\{a,b\}$ qui ont $ab$ et $ba$ pour facteurs.
  \end{question}

\end{exercice}
